# Agon Protocol

## Prerequisites

- Node.js 18+
- Rust
- Solana CLI
- Anchor

## Install

```bash
npm install
```

## Build

```bash
anchor build
```

## Test

```bash
anchor test
```

Windows + WSL local validator flow:

```powershell
.\scripts\test-local.ps1
```

## Verify

```bash
npm run verify
```

## Deploy

```bash
npm run deploy:devnet
```
